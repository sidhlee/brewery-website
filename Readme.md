# Brewery Website

## Setting up with Parcel

1. `yarn init --y`
2. `yarn add parcel-bundler --dev`
3. `package.json`

```json
 "scripts": {
    "dev": "parcel src/index.html",
    "build": "parcel build src/index.html"
  }
```

4. `src/index.html`

```html
<script src="index.js"></script>
</body>
```
